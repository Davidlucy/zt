# 张婷简历网站

这是一个静态网页项目，`index.html` 作为固定入口页，具体简历页面统一放在 `resumes/` 目录，便于后续扩展多份简历。

## 项目结构

- `index.html`：网站固定入口页（标题 + 入口按钮）
- `resumes/电商运营.html`：张婷电商运营方向简历内容页
- `img/`：个人照片等静态图片资源目录（例如 `img/个人照.jpg`）
- `JD/`：各公司岗位 JD 资料目录（当前含 `JD/转转电商/`）
- `蔚来知识/`：蔚来实习期间的工作记录与复盘截图素材
- `LICENSE`：开源许可证（MIT）

## 功能说明

- 首页展示“张婷的简历”和入口按钮
- 点击按钮 `电商运营` 后，跳转到 `resumes/电商运营.html`
- 简历内容包含：个人总结、教育经历、实习经历、社团和组织经历、专业技能

## 扩展规范（新增简历）

- `index.html` 尽量保持不变，只新增入口按钮
- 每一份简历单独放到 `resumes/` 目录，例如：`resumes/产品运营.html`
- 图片统一放在 `img/` 目录，简历页内通过相对路径引用（如 `../img/个人照.jpg`）
- `JD/` 用于按公司归档岗位要求，写简历时优先对齐其中关键词
- `蔚来知识/` 用于沉淀实习过程证据，写项目经历时优先使用可核验数据

## 本地预览

直接用浏览器打开 `index.html` 即可，或使用任意静态服务器：

```bash
python3 -m http.server 8080
```

然后访问：`http://localhost:8080`

## 部署说明

本项目为纯静态页面，可部署到任意静态托管平台，例如：

- GitHub Pages
- Netlify
- Vercel
- 腾讯云/阿里云静态网站托管

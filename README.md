# 张婷简历网站

这是一个静态网页项目，主页面提供一个入口按钮，点击后跳转到电商运营简历页面。

## 项目结构

- `index.html`：网站主界面（张婷的简历 + 入口按钮）
- `电商运营.html`：张婷电商运营方向简历内容页
- `LICENSE`：开源许可证（MIT）

## 功能说明

- 首页展示“张婷的简历”和入口按钮
- 点击按钮 `电商运营` 后，跳转到 `电商运营.html`
- 简历内容包含：个人总结、教育经历、实习经历、社团和组织经历、专业技能

## 本地预览

直接用浏览器打开 `index.html` 即可，或使用任意静态服务器：

```bash
python3 -m http.server 8080
```

然后访问：`http://localhost:8080`

## 部署说明

本项目为纯静态页面，可部署到任意静态托管平台，例如：

- GitHub Pages
- Netlify
- Vercel
- 腾讯云/阿里云静态网站托管
